# Madan Mina Sarvodaya Foundation (MAMI) Website

A professional NGO website with donation, events, gallery, and community management features.

## Features
- ✅ Responsive design with Tailwind CSS
- ✅ Donation system (with Razorpay support)
- ✅ Event management
- ✅ Photo gallery
- ✅ Configuration-driven setup
- ✅ Modular BAO/DAO architecture

## Quick Start

```bash
npm install
npm run dev
